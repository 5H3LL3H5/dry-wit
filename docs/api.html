arrayContains element array

Checks if an element is already contained in an array.

Parameters

element

array
The array (passed as “${array[@]}”).

Usage example

a=("one" "two" "three" "other number")
if arrayContains "other number" "${a[@]}"; then ...; fi

a=("one" "two" "three" "other number")
if arrayDoesNotContain "four" "${a[@]}"; then ...; fi

if getIndexOfItemInArray "a" "a b c"; then echo "Index of 'a': ${RESULT}"; then
